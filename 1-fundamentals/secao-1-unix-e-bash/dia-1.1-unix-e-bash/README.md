# Comandos unix

Comando          |Significado|                    
|----------------|-------------------------------|-----------------------------|
|`ls` |lista os arquivos e diretórios|
|`ls -la` |lista todos os arquivos e diretórios|
|`mkdir`|cria um diretório|
|`cd 'dir'`|altera para o diretório|
|`cd ~`|muda para o diretório `/home`|
|`cd ..`|retorna ao diretório 'pai'|
